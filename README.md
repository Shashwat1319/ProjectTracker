# Client Project Tracker
A simple web application to manage client projects and track activities in real-time.

## Features

- Add, view, update, and delete projects

- Track project status: Planned / In Progress / Completed

- Search and filter projects by client or name

- Add activity logs with timestamp

- Responsive design for desktop and mobile

## Technologies

- React

- Bootstrap

- JavaScript

- Node.js

- Express.js

- MongoDB

## Deployment

The project is live at: [Client Project Tracker](https://projectrackerr.netlify.app/)

## Installation
```bash
git clone https://github.com/Shashwat1319/Client-Project-Tracker.git

# Frontend
cd client
npm install
npm run dev

# Backend
cd server
npm install
npm run dev
```
## About

This project demonstrates a basic full-stack application suitable for managing client projects and logging activities.
